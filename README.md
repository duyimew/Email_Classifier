## Có thể xem web đã được deployed qua đường dẫn: 
https://emailclassifier-lab03.streamlit.app/

## Các bước chạy local:
### Bước 1: Clone repository
```bash
git clone https://github.com/duyimew/Email_Classifier.git
cd Email_Classifier
```
### Bước 2: Tải các requirements nếu chưa có
```bash
pip install -r requirements.txt
```
### Bước 3: Chạy ứng dụng web
```bash
streamlit run app.py
```
